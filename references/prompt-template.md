# 小红书爆款笔记完整输出模板

以下是每篇笔记的完整输出格式，严格按照此模板逐篇呈现。

## 输出格式

```
---
【第N篇】结构类型：{五种结构之一}

【标题】
{12-18字标题，不含标点符号}

【正文】
{300-600字正文内容，已通过AI味检测修正}

【关键词部署分析】
- 关键词清单：{关键词1}、{关键词2}、{关键词3}、{关键词4}、{关键词5}
- 关键词总出现次数：{正文中所有关键词出现总次数}
- 正文总字数：{实际字数}
- 关键词密度：{关键词总次数÷正文总字数×100}%（≥5%，≤10%）
- 部署说明：
  - 标题中出现的关键词：{列出}
  - 首段中出现的关键词：{列出}
  - 中间段落出现的关键词：{列出}
  - 结尾出现的关键词：{列出}
- 密度调整记录（若有）：{说明调整了哪些关键词的频次以达到目标密度}

【AI味检测分析】
- 检测项一：连接词清理 —— 检出{数量}处，已修正{数量}处
  - 修正对照：{原文} → {修改后}
- 检测项二：夸张修饰清理 —— 检出{数量}处，已修正{数量}处
  - 修正对照：{原文} → {修改后}
- 检测项三：推销语气转换 —— 检出{数量}处，已修正{数量}处
  - 修正对照：{原文} → {修改后}
- 检测项四：被动句转主动句 —— 检出{数量}处，已修正{数量}处
  - 修正对照：{原文} → {修改后}
- 检测项五：口语化表达强化 —— 已添加{数量}处
  - 添加位置：{说明在哪些句中添加了什么口语词}
- 综合AI味评分：{1-5分}
  - 5分=完全无AI味，自然流畅如真人分享
  - 4分=偶有1-2处轻微痕迹，整体自然
  - 3分=部分段落有AI感但可接受
  - 2分=多处明显AI痕迹
  - 1分=AI味明显，需重写
- 调整建议：{一句针对性建议，如仍有残留AI味的改进方向}

【合规自检】
- 结果：{通过/不通过}
- 若不通过，违规项：{列出具体违规内容和违反的规则}
---
```
